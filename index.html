Usage

/
returns HTML usage documentation
/api/v1/bgp/
returns $REMOTE_ADDR prefix and source asn in format "prefix $prefix advertised by $asn ($asname)"
/api/v1/bgp/<IP>
returns $IP prefix and source asn in format "prefix $prefix advertised by $asn ($asname)"
/api/v1/bgp/asn/
returns $REMOTE_ADDR source asn in format "$asn"
/api/v1/bgp/asn/<IP>
returns $IP source asn in format "$asn"
/api/v1/bgp/prefix/
returns $REMOTE_ADDR prefix in format "$prefix"
/api/v1/bgp/prefix/<IP>
returns $IP prefix in format "$prefix"
